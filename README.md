# Marketsquare

A full stack e-commerce website.

## Design

Check out the design on [Dribbble](https://dribbble.com/shots/21591624-Marketsquare)

## Functionalities

Functionalities include:

1. Product Catalog
2. Product Details
3. Shopping Cart
4. Checkout
5. Stripe Payments
6. Purchase Summary
